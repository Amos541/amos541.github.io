# 前言

在安装部署 Hexo 之后，目前的博客页面还是相当简洁。Hexo 作为一款优秀的静态站点生成器，提供了丰富的主题和插件供你选择，以满足不同的审美和功能需求。在这里，我将以本站所使用的 "Shoka" 主题为例，详细介绍其安装和配置过程。当然，如果你对其他主题或自定义主题感兴趣，也是完全可行的；不过每个主题的配置和所需插件可能会有所不同，这就需要你根据具体情况自行探索了。

再次感谢 Ruri Shimotsuki 大佬的 Shoka 主题，究竟是什么样的佬才能同时兼顾计算机和医学这两个领域？想想就让人觉得害怕
