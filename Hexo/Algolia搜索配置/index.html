# 前言

Algolia 是一个基于云的搜索引擎，它提供了丰富的功能，如搜索、自动补全、实时更新、个性化搜索、数据分析等。

第一次使用 shoka 主题时，点击右上角的搜索按钮，发现无法使用，这是因为还没有配置好 Algolia 插件，今天我们来配置下 Algolia 插件。

# 配置过程

# 获取 API Keys

  1. 打开 Algolia 官网，注册一个账号并登录。
  2. 点击仪表盘的左侧菜单栏的 search，进入搜索管理页面。

    点击创建索引。

  3. 创建完成之后点击左下角设置按钮，进入设置页面。点击 API Keys，进入 API Keys 页面。

    选择第二个选项，去创建一个新的 API Key。注意要选择 search 权限。


  4. 复制 Your API Keys 中的 Application ID 、 Admin API Key 和刚刚创建的 All API Keys 中的 Search API Key 。
