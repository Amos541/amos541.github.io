# 数据结构基本概念

  1. 数据
    • 数据是能输入计算机并且能被计算机处理的各种符号的集合
    • 数据是信息的载体
    • 数据是对客观事物符号化的表示
    • 数据能够被计算机识别、存储和加工
  2. 数据元素（Data Element）
    • 数据元素是数据的基本单位，通常作为一个整体进行考虑和处理
  3. 数据项（Data Item）
    • 一个数据元素由若干个数据项组成
    • 数据项是数据元素的不可分割的最小单位
  4. 数据对象（Data Object）
    • 数据对象是具有相同性质的数据元素的集合，是数据的一个子集
  5. 数据结构（Data Structure）
    • 数据结构是相互之间存在的一种或多种特定关系的数据元素的集合

# 数据结构三要素

  • 逻辑结构：从逻辑关系上描述数据，与数据的存储无关，它是从具体问题抽象出来的数学模型。
  • 物理结构（存储结构）：数据元素及其关系在计算机中内存中的存储形式（又称映像）。
  • 数据的运算：数据上的某些操作，包括：增删改查排序等。

# 逻辑结构的种类：划分方法一

线性结构：
- 有且仅有一个开始和一个终端结点，并且所有结点都最多只有一个直接前驱和一个后继。
- 例如：线性表、栈、队列、串
非线性结构：
- 一个结点可能有多个直接前驱和直接后继。
- 例如：树、图

# 逻辑结构的种类：划分方法二

集合：
- 数据元素之间除了 “同属于一个集合” 之外，没有其他关系。
线性结构：
- 一个对一个的线性关系，如线性表、栈、队列
树形结构：
- 一个对多个的层次关系
图形结构：
- 多个对多个的任意关系

# 物理结构（存储结构）的分类

顺序存储结构
链式存储结构
索引存储结构，如汉语字典
散列存储结构（哈希），通过计算之后找到指定位置

# 抽象数据类型

  1. 抽象数据类型定义
    • 数据类型：是一个值得集合和定义在此值集上得一组操作的总称。
      按 “值” 的不同特性数据类型可分为（高级程序语言）：
      (1) 原子类型：其值不可再分的数据类型
      C 语言：char、int、float、double、void
      (2) 结构类型：其值可再分解为若干成分的数据类型
