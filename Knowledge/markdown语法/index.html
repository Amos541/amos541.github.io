markdown语法

Amos Yun2024-08-012025-04-08

记录一下markdown语法

标题

  1. 使用 = 和 - 标记一级和二级标题。
  2. 使用 # 号标记。
    使用 # 号可表示 1-6 级标题，一级标题对应一个 # 号，二级标题对应两个 # 号，以此类推。

段落

  1. 段落的换行是使用两个以上空格加上回车。
  2. 在段落后面使用一个空行来表示重新开始一个段落。

字体

  1. 斜体文本
    斜体文本
  2. 粗体文本
    粗体文本
  3. 粗斜体文本
    粗斜体文本

分隔线
* * *

  1. 在一行中用三个以上的星号、减号、底线来建立一个分隔线，行内不能有其他东西。
  2. 在星号或是减号中间插入空格。

删除线
如果段落上的文字要添加删除线，只需要在文字的两端加上两个波浪线 ~~ 即可

下划线
下划线通过 HTML 的 u 标签来实现

脚注
Markdown 脚注的格式为[^要注明的文本]
这是一个注释[^注释]
[^注释]:这是一个注释。

列表

  1. 有序列表
    1. 这是第一个
    2. 这是第二个
  2. 无序列表
    • 这是第一个
    • 这是第二个

区块
Markdown 区块引用是在段落开头使用 > 符号 ，然后后面紧跟一个空格符号

代码
如果是段落上的一个函数或片段的代码可以用反引号把它包起来。

链接

  1. 使用[链接名称](链接地址)：百度
  2. <链接地址>：<www.baidu.com>

图片
插入图片使用![图片名称](图片链接)的格式：

表格
Markdown 制作表格使用 | 来分隔不同的单元格，使用 - 来分隔表头和其他行。
